About the Role

We're looking for a talented Full Stack Web Developer to join our team and build scalable, user-friendly web applications. You'll work with modern technologies and collaborate with designers, product managers, and other developers to deliver innovative solutions for diverse clients.

Responsibilities

  • Develop responsive web applications using React, Vue, or Angular
  • Build robust backend services using Node.js, Python, or Java
  • Design and optimize database schemas (SQL and NoSQL)
  • Implement RESTful APIs and GraphQL endpoints
  • Write clean, maintainable, and well-tested code
  • Collaborate with cross-functional teams to define requirements
  • Participate in code reviews and continuous improvement
  • Deploy and monitor applications on cloud platforms

Required Qualifications

  • 3+ years of professional web development experience
  • Strong proficiency in JavaScript/TypeScript
  • Experience with modern frontend frameworks (React, Vue, Angular)
  • Backend development experience (Node.js, Python, Java, or similar)
  • Database design and SQL/NoSQL knowledge
  • Understanding of RESTful APIs and web services
  • Version control (Git) and CI/CD pipelines
  • Problem-solving skills and attention to detail

Preferred Qualifications

  • Experience with cloud platforms (AWS, Google Cloud, Azure)
  • Knowledge of containerization (Docker, Kubernetes)
  • Experience with testing frameworks and TDD practices
  • Familiarity with GraphQL and microservices architecture
  • Open-source contributions or personal projects
  • Agile/Scrum development experience

What We Offer

  • Comprehensive health, dental, and vision coverage
  • 401(k) matching (6% match)
  • Unlimited PTO policy
  • Flexible work arrangements (hybrid available)
  • Professional development and conference budget
  • Modern development tools and hardware
  • Collaborative and innovative work environment
