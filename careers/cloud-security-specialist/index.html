About the Role

Join our Cloud Security team to design, implement, and maintain robust security architectures protecting enterprise infrastructure and sensitive data. You'll work with Fortune 500 companies to secure their cloud environments across AWS, Azure, and Google Cloud platforms.

Responsibilities

  • Design and implement cloud security architectures for enterprise clients
  • Conduct security assessments and penetration testing
  • Configure and manage cloud security tools (firewalls, IDS/IPS, SIEM)
  • Develop security policies and compliance frameworks
  • Respond to security incidents and implement remediation strategies
  • Stay current with emerging cloud security threats and best practices
  • Collaborate with DevOps and infrastructure teams
  • Create security documentation and conduct training sessions

Required Qualifications

  • 4+ years of professional cloud security experience
  • Deep knowledge of AWS, Azure, or Google Cloud security services
  • Understanding of network security, firewalls, and VPNs
  • Experience with security compliance frameworks (SOC 2, HIPAA, PCI-DSS)
  • Strong knowledge of encryption and cryptography
  • Proficiency in Linux and/or Windows administration
  • Relevant certifications (CCNA, CEH, OSCP, or cloud certifications)
  • Excellent communication and problem-solving skills

Preferred Qualifications

  • AWS Solutions Architect or Security Specialty certification
  • Experience with infrastructure-as-code and CloudFormation
  • Knowledge of containerization and Kubernetes security
  • Experience with SIEM tools (Splunk, ELK, Datadog)
  • Incident response and forensics experience
  • Background in threat intelligence or vulnerability management

What We Offer

  • Competitive salary: $150,000 - $210,000/year
  • 100% remote work with flexible hours
  • Comprehensive health, dental, and vision coverage
  • 401(k) matching (6% match)
  • Unlimited PTO policy
  • Professional development and certification budget
  • Quarterly team offsites
  • Stock options for senior roles
